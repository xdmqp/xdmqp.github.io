Linux常用命令

2020-07-12
  • Linux

ls：表示列出当前目录下的文件，ls的命令补充如下：

​ -l：列出文件详细信息
​ -t：以时间排序，默认最新修改的文件在上面
​ -r：倒序输出
​ -x：按列输出，横向排序
​ -u：按照文件上次被访问的时间排序
​ -lrt，代表了”-l -r -t”，按修改时间倒序列出当前目录下的文件

grep：正则表达式搜索文本

用法：grep [命令补充] pattern files
-r：搜索子目录
-n：显示匹配行和行号
-l：只列出匹配的文件名
-i：不区分大小写，默认区分大小写
-w：只匹配整个单词，而不是字符串的一部分
grep pattern1 | pattern2 files ：显示匹配 pattern1 或 pattern2 的行
grep pattern1 files | grep pattern2 ：显示既匹配 pattern1 又匹配 pattern2 的行

vim/vi：文件编辑

vi/vim 共分为三种模式，分别是命令模式（Command mode），输入模式（Insert mode）和底线命令模式（Last line mode）， 三种模式的切换方式如下图所示

命令模式下常用命令
  1. i:进入输入模式，从光标当前位置开始输入
  2. a:进入输入模式，从光标所在位置的下一个位置开始输入
  3. o：进入输入模式，插入新的一行，从行首开始输入
  4. 0:移动光标到文件开头
  5. G：移动光标到文件最后
  6. $：移动光标到所在行行尾
  7. ^：移动光标到所在行行首
  8. #l：移动光标到当前行的第#个位置
  9. x:删除光标所在位置后面的一个字符
  10. X：删除光标所在位置前面的一个字符
  11. dd：删除光标所在行
  12. #dd：从光标所在行开始删除#行
  13. ctrl+g：列出光标所在行行号
  14. #G：移动到光标只文件的第#行行首
  15. u：撤销操作
  16. ctrl+r：恢复操作
底线命令模式下常用命令
  1. set nu：显示文件行号
  2. #：跳到文件中的第#行
  3. /关键字：查找指定关键字，按n会继续往后查找
  4. w：保存文件
  5. q：退出文件编辑
  6. wq：保存并退出
  7. q!：放弃一切更改，强制退出
