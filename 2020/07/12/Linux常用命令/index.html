Linux常用命令

ls命令

用法：ls [option]，列出当前目录下的文件

-l：列出文件详细信息

-a :列出目录所有文件，包含以.开始的隐藏文件

-t：以时间排序，默认最新修改的文件在上面

-r：倒序输出

-x：按列输出，横向排序

-u：按照文件上次被访问的时间排序

-S：以文件大小排序

-h：以易读大小显示

组合命令，如-lrt，代表了”-l -r -t”，按修改时间倒序列出当前目录下的文件

grep

用法：grep [option] pattern files，正则表达式搜索文本

-r：搜索子目录

-n：显示匹配行和行号

-l：只列出匹配的文件名

-i：不区分大小写，默认区分大小写

-w：只匹配整个单词，而不是字符串的一部分

grep pattern1 | pattern2 files ：显示匹配 pattern1 或 pattern2 的行

grep pattern1 files | grep pattern2 ：显示既匹配 pattern1 又匹配 pattern2 的行

vim/vi：文件编辑

vi/vim 共分为三种模式，分别是命令模式（Command mode），输入模式（Insert mode）和底线命令模式（Last line mode）， 三种模式的切换方式如下图所示

命令模式下常用命令
  1. i:进入输入模式，从光标当前位置开始输入
  2. a:进入输入模式，从光标所在位置的下一个位置开始输入
  3. o：进入输入模式，插入新的一行，从行首开始输入
  4. 0:移动光标到文件开头
  5. G：移动光标到文件最后
  6. $：移动光标到所在行行尾
  7. ^：移动光标到所在行行首
  8. #l：移动光标到当前行的第#个位置
  9. x:删除光标所在位置后面的一个字符
  10. X：删除光标所在位置前面的一个字符
  11. dd：删除光标所在行
  12. #dd：从光标所在行开始删除#行
  13. ctrl+g：列出光标所在行行号
  14. #G：移动到光标只文件的第#行行首
  15. u：撤销操作
  16. ctrl+r：恢复操作
底线命令模式下常用命令
  1. set nu：显示文件行号
  2. set nonu：隐藏文件行号
  3. #：跳到文件中的第#行
  4. /关键字：查找指定关键字，按n会继续往后查找
  5. w：保存文件
  6. q：退出文件编辑
  7. wq：保存并退出
  8. q!：放弃一切更改，强制退出

cd命令

用法 cd [option] path

/path:切换到根目录中的path目录下

./path：切换到当前目录的path目录下

../path:切换到上层目录的path目录下

~：切换到当前用户的自家目录下，如果为root用户，就进去root目录下

-：切换到进入当前目录的前的目录下

pwd命令

打印当前工作目录路径

-L：打印逻辑工作目录路径

-P：打印物理工作目录路径

mkdir命令

用法：mkdir [option] dirname，创建文件夹

-m：对创建文件夹设置存取权限

-p：在指定目录下创建文件夹

rm命令

用法：rm [option] name，删除文件或者目录

-i ：删除前逐一确认

-f：直接删除，无需确认

-r：删除指定目录及以下的所有文件

rmdir命令

用法： rm [-p] name，删除空目录

rz命令

用法：rz [option]，使用ZMODEM协议将本地文件上传到远程Linux服务器，不能操作文件夹

-+, –append:将文件内容追加到已存在的同名文件

-a,–ascii:以文本方式传输

-b, –binary:以二进制方式传输，推荐使用

–delay-startup N:等待N秒

-e, –escape:对所有控制字符转义，建议使用

-E, –rename:已存在同名文件则重命名新上传的文件，以点和数字作为后缀

-p, –protect:对ZMODEM协议有效，如果目标文件已存在则跳过 -

q, –quiet:安静执行，不输出提示信息

-v, –verbose:输出传输过程中的提示信息

-y, –overwrite:存在同名文件则替换

-X, –xmodem:使用XMODEM协议

–ymodem:使用YMODEM协议

-Z, –zmodem:使用ZMODEM协议

–version：显示版本信息

–h, –help：显示帮助信息

sz命令

用法：sz [option] [filelist]，使用ZMODEM协议将多个文件从远程服务器下载到本地，不能操作文件夹

示例：sz file1 file2 file3

mv命令

用法：mv [option]

命令 操作
mv 文件名 文件名 将源文件名改为目标文件名
mv 文件名 目录名 将文件移动到目标目录
mv 目录名 目录名 目标目录已存在，将源目录移动到目标目录；目标目录不存在则改名

-i: 若指定目录已有同名文件，则先询问是否覆盖旧文件

-f: 在 mv 操作要覆盖某已有的目标文件时不给任何指示
